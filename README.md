# Lista de Exercícios de Condicionais em C

Este projeto em C demonstra o uso de estruturas de controle de fluxo (`if`, `else if`, `else` e `do-while`) para resolver uma série de exercícios de lógica. O programa interativo permite ao usuário selecionar diferentes funcionalidades através de um menu.

## Funcionalidades

* **Soma e Comparação:** Verifica se a soma de dois valores é menor que um terceiro.
* **Par ou Ímpar:** Identifica se um número inserido é par ou ímpar.
* **Operações Condicionais:** Soma dois valores se forem iguais ou os multiplica caso sejam diferentes.
* **Dobro ou Triplo:** Calcula o dobro de um número positivo e o triplo de um negativo.
* **Cálculo Condicional:** Soma 5 a um número par ou 8 a um ímpar.
* **Cálculo de IMC:** Determina o Índice de Massa Corporal e sua classificação (peso ideal, sobrepeso, etc.).
* **Sair:** Encerra o programa de forma segura.
